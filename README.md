## 2. Description
This is a landing page mockup created by replicating a design directly from a Figma file, aiming for the highest possible fidelity to the original design.

## 3. Table of Contents
The page is structured vertically, and navigation between different sections is handled via scrolling. It consists of:

*   **Header:** Includes the Navbar for navigation.
*   **Main:** Contains a total of **8 sections** arranged sequentially.
*   **Footer:** Page closure with complementary information.
*   **Interactivity:** The project features transformations on buttons and specific images, along with `hover` effects and transitions applied to interactive elements.

## 4. Project Structure
The file organization is as follows:

*   `index.html`: Main file with the page structure.
*   `style.css`: Stylesheet containing all visual configurations.
*   `README.md`: Project documentation.
*   `/assets`: Folder containing all images and visual resources used.
